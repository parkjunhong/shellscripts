## 'jump'에 앞서 확인해야 할 사항.
- 현재 연결정보 구분은 어떻게 하지?
  => 신규 'id' 항목을 추가
- 'jump' 참조연결정보가 'jump'를 가지고 있는 경우는 어떻게 처리가 되지?
  - 참조구조 예)
    ```
    - A
      - jump: [B, C, D]
    - B 
      - jump: [b1, b2]
    - C
    - D
      - jump :[d1, d2]
    ```
  - 'connect' A 를 하면, 과정은 어떤 게 합리적일까?
  => A -> B -> b1 -> b2 -> C -> D -> d1 -> d2
