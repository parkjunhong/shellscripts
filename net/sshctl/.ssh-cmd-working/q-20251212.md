    ```
    - A
      - jump: [B, C, D]

    - C
    - D
      - jump :[d1, d2]
    ```
  - 'connect' A 를 하면, 과정은 어떤 게 합리적일까?
  => A -> B -> b1 -> b2 -> C -> D -> d1 -> d2
